Reat e memoização, quando utilizar e quando não utilizar Há componentes memoizados que continuam a renderizar, que são os tipos não escalares. O que são tipos escalares vs não escalares? - Tipos escalares: string, number, boolean, null, undefined, symbol. São comparados por valor, ou seja, se o valor for o mesmo, eles são considerados iguais. - Tipos não escalares: objetos, arrays, funções. São comparados por referência, ou seja, mesmo que o conteúdo seja o mesmo, se forem instâncias diferentes, eles são considerados diferentes.
